# Movie App

A movie search app built with HTML, CSS, and JavaScript.

## Features
- Search movies by title
- Show posters, titles, and years
- Click a movie to see more details
- Display plot and IMDb rating
- Loading and error handling

## Technologies Used
- HTML
- CSS
- JavaScript
- OMDb API

## How to Run
1. Download or clone the project
2. Open `index.html` in your browser
